
# mytasks secure Symfony


![Logo](https://symfony.com/logos/symfony_white_02.png)




This project is a secure task management web application built with the Symfony framework.
It allows users to create, update, delete, and toggle the status of their personal tasks
while ensuring proper authentication and authorization.
